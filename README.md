# ✅ Taskify

A responsive **Task Management** web application built with **React.js** that helps users organize and manage their daily tasks efficiently. Users can add tasks, assign priorities and categories, mark tasks as completed, track their progress, and store tasks using Local Storage for a seamless experience.

---

## 🚀 Features

* ✅ Add new tasks
* ✅ Delete tasks
* ✅ Mark tasks as Complete / Undo
* ✅ Set task priority (High, Medium, Low)
* ✅ Assign task categories (General, Work, Personal)
* ✅ Track task completion with a progress bar
* ✅ Clear all tasks
* ✅ Local Storage support
* ✅ Responsive and user-friendly interface

---

## ▶️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/Deepa-noolvi/Taskify.git
```

### 2. Navigate to the project folder

```bash
cd Taskify
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

```bash
npm start
```
